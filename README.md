# Smart Interview Coach Pro

## Overview

Smart Interview Coach Pro is a Python-based console application designed to help users practice technical interview questions and assess their readiness through an interactive scoring system.

This project was developed as part of my Python and AI learning journey to strengthen programming fundamentals, problem-solving skills, and application development.

## Features

* Interactive interview experience
* Multiple Python-related questions
* Instant answer validation
* Real-time score tracking
* Percentage calculation
* Final performance assessment
* Interview readiness feedback

## Technologies Used

* Python
* Conditional Statements
* Variables
* User Input Handling
* Score Calculation Logic

## How It Works

1. The user starts the interview session.
2. The application asks technical Python questions.
3. User responses are evaluated.
4. Scores are tracked throughout the session.
5. A final report is generated with percentage and performance level.
